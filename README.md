# 🛒 Supermarket Billing System (C++)

A console-based Supermarket Billing System developed using **C++** and **Object-Oriented Programming (OOP)** concepts. This project allows users to manage products, generate bills, and store product information permanently using file handling.

---

## 📌 Features

- ➕ Add Products
- 📋 Display All Products
- 🔍 Search Products
- ✏️ Update Product Details
- ❌ Remove Products
- 🧾 Generate Bills
- 📦 Automatic Stock Management
- 💾 Save Products to File
- 📂 Load Products Automatically
- 🏛️ Inheritance
- 🎭 Runtime Polymorphism

## 🚀 How to Run

### Compile

```bash
g++ main.cpp Product.cpp Store.cpp DiscountProduct.cpp -o Supermarket
```

### Run (Windows PowerShell)

```powershell
.\Supermarket.exe
```
